World - Countries Map (Shape format)  
File Information: Name: world_countries_shp.zip Size: 976.99 KB Date: 08/07/2002

We are pleased to offer you this FREE World dataset in ESRI Shape format, including comprehensive attributes in dBase (DBF) format. Please feel free to use this data in your application.

Features included: Country boundaries.
Attributes included: Continent, Unique ID, Capital city, 1994 population, ISO numeric code, ISO standard abbreviation, Three-letter abbreviation, Complete country name.
